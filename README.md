# 🌡️ 공간 데이터 기반 폭염 취약 계층을 위한 쿨링포그 최적 입지 분석

## 📌 프로젝트 개요

본 프로젝트는 기후위기로 인해 심화되는 폭염 문제에 대응하고,
특히 고령층과 같은 폭염 취약 계층을 보호하기 위해 수행되었습니다.

기존의 폭염 대응 시설은 단순 분산 설치 방식으로 운영되는 경우가 많아
실제 수요와의 불일치 문제가 존재합니다.

이를 해결하기 위해 공간 데이터와 생활 인프라 데이터를 결합하고,
최적화 모델(MCLP, P-median)을 적용하여
**데이터 기반 쿨링포그 설치 최적 입지**를 도출하였습니다. 

---

## 🎯 문제 정의

* 폭염 취약 계층(고령층)의 생활권을 고려한 시설 배치 부족
* 기존 폭염 대응 정책의 데이터 기반 의사결정 부족
* 제한된 설치 개수 내에서 최대 효과를 내는 입지 선정 필요

👉 **“어디에 설치해야 가장 많은 취약 계층을 효과적으로 보호할 수 있는가?”**

---

## 🔍 분석 접근 방법

본 프로젝트는 **공간 최적화 문제**로 정의하고,
두 가지 대표적인 입지 선정 모델을 결합하여 해결하였습니다.

### ✔️ MCLP (Maximum Covering Location Problem)

* 목적: 최대 수요 커버
* 반경 내 포함되는 수요지 최대화

### ✔️ P-median

* 목적: 수요지와 시설 간 거리 합 최소화
* 접근성 중심 최적화

👉 두 모델 결과의 **공통 입지**를 최종 후보지로 선정 

---

## 📊 사용 데이터

* **인구 데이터**: 65세 이상 고령 인구 분포
* **수요지 데이터**:

  * 경로당, 노인시설, 노인복지시설
  * 버스 정류장 (유동 인구 고려)
* **후보지 데이터**:

  * 공원, 무더위 쉼터, 야외 쉼터, 운동시설
* **기타 데이터**:

  * 기온, 폭염일수, 교통 이용량 등

📌 다양한 공공데이터 및 API 기반 수집 

---

## ⚙️ 분석 프로세스

1. **데이터 수집**

   * 공공데이터포털, 기상청, 교통 데이터 등 활용

2. **데이터 전처리**

   * 주소 → 위경도 변환 (지오코딩)
   * 행정동 기준 데이터 통합
   * 고령층 인구 기준 분석 대상 선정

3. **분석 대상 선정**

   * 노인 인구 상위 3개 행정동:

     * 청룡동, 신안동, 성환읍 

4. **모델링**

   * Haversine 기반 거리 계산
   * MCLP / P-median 모델 적용

5. **결과 병합**

   * 두 모델 공통 입지 추출

---

## 📍 주요 결과

* 천안시 노인 인구 상위 3개 지역 중심 분석 수행
* 두 최적화 모델을 결합하여 **최종 5개 입지 도출**

### ✔️ 최종 선정 입지

* 청룡동: 청당코오롱하늘채, 하천변
* 신안동: 신부문화공원, 신부6공원
* 성환읍: 이화아파트 인근
